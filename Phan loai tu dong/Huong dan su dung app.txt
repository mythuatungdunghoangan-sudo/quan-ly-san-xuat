Ứng dụng giúp bóc tách đơn hàng tự động từ file PDF, ảnh chụp,
hoặc Excel của khách hàng, rồi tổng hợp vào file kế hoạch sản
xuất Excel (6 sheet: Nhãn C115 · Nhãn Decan · Hộp · Thùng · Túi màng · Tổng hợp).


------------------------------------------------------------
  TRUY CẬP ỨNG DỤNG
------------------------------------------------------------

  Qua mạng (không cần cài gì):
  → quan-ly-san-xuat-4wnwfzg9zujxcjnfmptdq4.streamlit.app

  Trên máy tính (offline):
  → Double-click file  Phan loai tu dong.bat


------------------------------------------------------------
  CÀI ĐẶT LẦN ĐẦU (máy mới — chỉ làm 1 lần)
------------------------------------------------------------

  Bước 1: Copy thư mục QuanLySanXuat vào máy
           (qua USB, OneDrive, hoặc tải từ GitHub)

  Bước 2: Vào thư mục "Phan loai tu dong"
           Double-click  cai_dat.bat
           → Tự cài Python nếu chưa có (cần internet ~27MB)
           → Tự cài toàn bộ thư viện cần thiết
           → Chờ 2–5 phút

  Bước 3: Double-click  Phan loai tu dong.bat  để mở app

  Từ lần sau: chỉ cần  Phan loai tu dong.bat  là đủ.


------------------------------------------------------------
  CÁCH SỬ DỤNG HÀNG NGÀY
------------------------------------------------------------

  1. Mở app (Phan loai tu dong.bat hoặc link cloud)

  2. Nhập Claude API Key vào sidebar (tùy chọn, nên có)
     - Không có key: chỉ đọc được PDF có text và Excel
     - Có key:       đọc được PDF scan, ảnh chụp tay
     - Key dạng: sk-ant-...
     - Mỗi lần mở app phải nhập lại (không lưu)

  3. Bấm "Tạo / tải template mặc định" (lần đầu)
     → Tải file Excel mẫu về xem cấu trúc

  4. Bấm Upload → chọn file đơn hàng từ khách hàng
     Hỗ trợ: PDF · PNG · JPG · Excel (.xlsx, .xls)
     Có thể upload nhiều file cùng lúc (nhiều công ty)

  5. Kiểm tra bảng dữ liệu — sửa trực tiếp nếu cần
     Cột "Loại" có thể đổi: Nhãn / Hộp / Thùng / Túi màng

  6. Bấm "Tổng hợp và tải về Excel"
     → File kế hoạch sản xuất tự động phân theo sheet


------------------------------------------------------------
  CẬP NHẬT APP SAU KHI CHỈNH SỬA CODE
------------------------------------------------------------

  Sau khi nhờ Claude Code chỉnh sửa, nói:
  "đẩy code lên GitHub"
  → Claude tự chạy lệnh, cloud cập nhật sau 1–2 phút

  Hoặc tự chạy trong PowerShell (thư mục QuanLySanXuat):
    git add .
    git commit -m "mô tả thay đổi"
    git push


------------------------------------------------------------
  CÀI ĐẶT MÔI TRƯỜNG LẬP TRÌNH (máy mới)
------------------------------------------------------------

  Nếu muốn tiếp tục phát triển app bằng Claude Code:

  1. Cài Git        → git-scm.com/download/win
  2. Cài Python     → python.org/downloads
                      (nhớ tick "Add Python to PATH")
  3. Cài Node.js    → nodejs.org  (bản LTS)
  4. Cài Claude Code → mở PowerShell, gõ:
                       npm install -g @anthropic-ai/claude-code
  5. Tải code về:
     git clone https://github.com/mythuatungdunghoangan-sudo/quan-ly-san-xuat.git QuanLySanXuat
  6. Cài thư viện:
     cd "QuanLySanXuat\Phan loai tu dong"
     pip install -r requirements.txt
  7. Mở Claude Code:
     cd QuanLySanXuat → gõ: claude
     Nói: "đọc CLAUDE.md để tiếp tục"
